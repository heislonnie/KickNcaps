KickNcaps - Premium Snapback Store

KickNcaps is a modern, sleek e-commerce demo built with React, Tailwind CSS, and Framer Motion. It showcases a premium snapback collection with interactive features including a sliding product carousel, a dynamic shopping cart, and a simulated payment flow.

This project is designed to demonstrate clean, modern frontend development with a polished UI/UX that feels production-ready.

Features

Responsive Layout – Works seamlessly on mobile, tablet, and desktop.

Modern Navbar – Fixed, glass-like navbar with hover effects and cart badge.

Interactive Product Slider – 11+ images, continuous sliding animation, 3D hover effect.

Dynamic Shopping Cart

Add, remove, increment, decrement items

Persistent via localStorage

Shows total items and price

Sleek PayButton Component

Validates email before payment

Smooth “Processing” and “Paid” states

Confetti animation on success

Reusable Components – Fully modular for easy scaling.

Tailwind CSS – Modern styling with responsive design and hover animations.

Framer Motion – Smooth animations for sliders and interactions.

Tech Stack

Frontend: React, Tailwind CSS, Framer Motion

State Management: React Context API (CartContext)

Animations: Framer Motion, react-confetti

Routing: React Router DOM

Persistence: LocalStorage for cart state
